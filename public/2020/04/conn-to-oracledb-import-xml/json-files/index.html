XML Parsing

  • html parsing과 동일
  • xml은 엄격한 HTML, 태그 해석을 브라우저가 하지 않고 클라이언트가 직접한다는 점이 HTML과 다른 점
  • 모든 HTML parsing은 XML에 적용 가능
  • XML parsing 방법으로는 parsing하지 못하는 HTML이 있을 수도 있음
